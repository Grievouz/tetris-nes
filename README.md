# Tetris NES Version
It'a just Tetris nothing more, nothing less. I only uploaded it because it was one of my **first** Projects.

## Controls
- Press **W,A,S,D** to **move** the tetromino
 - Press **S** to **restart** the game
 - Press **P** to **pause** the game
